# xlsx to vcf



## 特性

能够把 xlsx 文件 导出成 vcf 通信录文件

#### 格式

xlsx 文件需要安装一下格式：

1. A 列为电话号码
2. B 列为联系人名称

示例如下：

|      | A           | B    |
| ---- | ----------- | ---- |
| 1    | 13710607594 | A君  |
| 2    | 13710607595 | B君  |
| 3    | 13710607596 | C君  |



## 开发

#### 安装依赖

```
yarn
```

#### 运行

```
yarn start
```

#### 打包成exe

```
yarn exe
```
